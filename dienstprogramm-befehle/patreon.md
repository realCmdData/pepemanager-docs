# Patreon

### Beschreibung

Dieser Befehl kann verwendet werden, um den Patreon-Link von Pepe Manager in einer netten kleinen Embed-Box zu senden, die nur für deine Augen gemacht ist ╰\(_°▽°_\)╯

### Befehlstruktur

```text
p!patreon
```

### **Berechtigungen**

* N/A **\(Benutzer\)**
* `Links einbetten`**\(Bot\)**

### Aliase

* `patreon`
* `donate`
* `patron`
