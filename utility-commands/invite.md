# Invite

### Description

This command can be used to send an invite link to the Pepe Manager bot in a nice little embed box, bespoke made just for your eyes ╰\(\*°▽°\*\)╯

### Command Structure

```text
p!invite
```

### **Permission**

* N/A **\(User\)**
* Embed Links **\(Bot\)**

### Aliases

* `invite`
* `inv`
