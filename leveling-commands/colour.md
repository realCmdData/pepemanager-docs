# Colour

### Description

This command can be used to change the colour of the progression bar in the [level ](level.md)command. The colour change is only for the server you're currently using the command in

### Command Structure

```text
p!leaderboard
```

### **Permission**

* N/A **\(User\)**
* N/A **\(Bot\)**

### Aliases

* `leaderboard`
* `lb`
